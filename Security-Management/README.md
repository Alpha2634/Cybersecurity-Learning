# Basics of Security Management

This section contains my learning notes from the Security Management course.
The focus is on understanding core security concepts used in organizations.

## Key Topics Learned

### CIA Triad
- Confidentiality: Protecting data from unauthorized access
- Integrity: Ensuring data is accurate and not altered
- Availability: Making sure systems and data are accessible when needed

### Security Policies
- Rules and guidelines that define how information security is managed
- Help organizations protect data and systems

### Risk Management
- Identifying security risks
- Analyzing their impact
- Taking steps to reduce or control risks

### Access Control
- Managing who can access systems and data
- Includes authentication and authorization concepts
